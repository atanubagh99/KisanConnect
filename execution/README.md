# Execution — Layer 3 (Deterministic Scripts & Utilities)

This directory contains **deterministic utility scripts** that handle repeatable tasks.

Scripts here should be:

- **Reliable** — produce consistent results given the same inputs
- **Testable** — can be verified independently
- **Idempotent** — safe to re-run without side effects (where possible)
- **Well-commented** — explain _why_, not just _what_

Supported languages: Java, Python, Shell scripts.
